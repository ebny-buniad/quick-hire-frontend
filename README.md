# Job Portal Frontend

A modern job portal frontend built with **Next.js**, **TypeScript**, and **Tailwind CSS**.  
This application allows users to explore jobs, filter and search positions, and view job details with a clean and responsive UI.

---

## 🚀 Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- pnpm (Package Manager)
- Lucide React Icons

---

## ✨ Features

### 🔎 Job Search
- Search jobs by **title**
- Search jobs by **location**

### 🎯 Advanced Filters
Users can filter jobs using:
- Employment Type (Full-time, Part-time, Internship, Contract)
- Workplace Type (Remote, Hybrid, Onsite)
- Experience Level (Junior, Mid, Senior)

### 📄 Job Listings
- Dynamic job cards
- Company logo and name
- Job category
- Location
- Employment type
- Experience level
- Salary range
- Job details navigation

### 🎨 Modern UI
- Responsive grid layout
- Consistent card footer alignment
- Clean Tailwind design
- Icon support with Lucide React

### 📑 Pagination
- Server-side pagination support
- Page-based navigation
